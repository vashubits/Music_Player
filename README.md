# 🎵 TuneX – Full‑Stack Music Player

## 📌 Overview
TuneX is a premium full‑stack music player web app built with **React, Express, and MongoDB**.  
It features a modern **pink/black gradient UI with glass‑effect cards**, seamless **frontend/backend sync**, and recruiter‑friendly polish for professional presentation.

## 🚀 Features
- 🎨 Modern UI with gradient theme and glass cards  
- 🔐 Secure user registration & role selection (dropdown/radio buttons)  
- ⚡ Error handling with clear frontend popups  
- 🎶 Music playback with playlist management  
- 🌐 Hosted frontend on **Vercel** and backend on **Render**  
- 🗄️ MongoDB integration for persistent storage  

## 🛠️ Tech Stack
- **Frontend:** React, CSS, Axios  
- **Backend:** Express.js, Node.js, MongoDB  
- **Hosting:** Vercel (frontend), Render (backend)  
- **Tools:** Docker, Postman for API testing
